# Information Search and Fact Collection

`information-search-and-fact-collection` covers:

Tasks that gather, verify, compare, and assemble factual information from live sources or public APIs into structured deliverables.

## Coverage Note

This is the strongest pattern in the current source sets. It includes open-ended research tasks and API-driven directory, weather, repo, and economic snapshots.

## Current Families

- `economic-snapshot`: Collect economic indicators for several countries and compare them. Variants: e1, m1, h1.
- `gitlab-project-analysis`: Analyze repository metadata and activity signals across multiple GitLab projects. Variants: e1, m1, h1.
- `market-research-dossier`: Survey a market segment and synthesize competitor facts into a concise research report. Variants: reference.
- `tech-conference-research`: Find live events and normalize them into a verified event list. Variants: reference.
- `university-directory-builder`: Build country-level university directories and summary statistics. Variants: e1, m1, h1.
- `weather-monitor`: Collect weather facts for multiple cities and summarize comparisons. Variants: e1, m1, h1.
